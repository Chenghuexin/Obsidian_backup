# Sentiment
- 定義 : 對某東西的評價
	- 表達方式 : 
		- 類別式 : Positive、Negative、Neutral
		- 數字型 : -5 ~ +5
		- 連續行 : [0, 1]
- valence : 喜歡程度
- arousal : 強度
